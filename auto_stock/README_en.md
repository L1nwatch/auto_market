# Background

Studying quantitative trading systems, whose core is strategy plus automation. Based on my own situation, I implemented code to automatically buy and sell stocks using the Tonghuashun client with a Ping An Securities account. Combined with a few simple strategies I wrote, it serves as a beginner's approach to quantitative trading.

# Strategy

I don't yet understand advanced trading strategies. For now I simply pick stocks that have fallen sharply recently and buy them. The cost price is recorded as x, and every day a sell order is placed at `x*1.03` for profit taking and `x*0.8` for stop loss.

# Running Instructions

* Environment: Windows 10 + Python 3.7 + Tonghuashun trading client 5.0

![Tonghuashun trading client 5.0](version_info.png)

* Steps:
    * Open the Tonghuashun trading client 5.0 and log in manually
    * After installing dependencies, run `main.py`
* Logic: the script loops continuously. Only during the time period **Monday to Friday, 8:30–9:00 AM** will the strategy run once to get that day's buy list, place the orders, and, according to the take-profit and stop-loss conditions, place sell orders for all available holdings.
